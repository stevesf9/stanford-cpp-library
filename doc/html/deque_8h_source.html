47 /*
48  * Class: Deque<ValueType>
49  * -----------------------
50  * This class models a linear structure called a <b><i>deque</i></b>
51  * in which values can be added and removed at either end.
52  * This discipline allows first-in/first-out (FIFO) and/or
53  * last-in/first-out (LIFO) behavior. That is the defining
54  * feature of deques.
55  */
