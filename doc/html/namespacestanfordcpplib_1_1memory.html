void computeMemoryDistances ( void *const  p,
unsigned long int &  stackDist,
unsigned long int &  heapDist,
unsigned long int &  staticDist 
)

Fills in the given long int parameters with the distance from the given pointer p to the stack, the heap, and the static storage area.

We work under the assumption that the smallest of these distances represents the area in which p was allocated.
