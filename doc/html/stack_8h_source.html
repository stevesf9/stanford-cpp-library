46 /*
47  * Class: Stack<ValueType>
48  * -----------------------
49  * This class models a linear structure called a <b><i>stack</i></b>
50  * in which values are added and removed only from one end.
51  * This discipline gives rise to a last-in/first-out behavior (LIFO)
52  * that is the defining feature of stacks. The fundamental stack
53  * operations are <code>push</code> (add to top) and <code>pop</code>
54  * (remove from top).
55  */
